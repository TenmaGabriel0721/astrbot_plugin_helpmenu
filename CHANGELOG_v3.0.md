# AstrBot HelpMenu 插件 v3.0.0 更新日志

## 🎨 二次元梦幻风格大升级

**发布日期**: 2025-12-15

---

## ✨ 主要改进

### 1. 图标系统全面升级

#### 🎭 梦幻风格图标生成器
- **3色径向渐变**: 从简单的2色线性渐变升级到3色径向渐变，层次更丰富
- **外发光效果**: 添加3层渐变光晕，营造梦幻氛围
- **多层高光**: 顶部高光 + 形状高光，增加立体感和光泽度
- **内阴影效果**: 形状内部添加柔和阴影，增强深度感
- **新增六边形**: 图标形状从5种增加到6种（圆形、方形、菱形、三角形、星形、六边形）

#### 🌈 配色方案优化
- 新增2套梦幻配色（珊瑚梦幻、薰衣草梦幻）
- 所有配色升级为3色渐变系统
- 颜色更柔和、更梦幻，符合二次元风格

### 2. 透明度系统优化

#### 📊 默认值调整（不遮挡背景人物）
- **分类背景透明度**: 从 8% 降低到 **3%**（几乎透明）
- **卡片透明度**: 从 90% 降低到 **70%**（半透明，背景清晰可见）
- **背景遮罩**: 从 1.5x 降低到 **0.35x**（大幅减少遮挡）

#### 🎨 配色层次优化
- **极淡主题色**: 98% 白色混合（原95%）
- **淡主题色**: 92% 白色混合（原85%）
- **中等主题色**: 85% 白色混合（原70%）
- **阴影透明度**: 降低到 10%（原15%），更柔和

### 3. 视觉效果增强

#### ✨ 梦幻风格特性
- 柔和的外发光效果
- 多层次渐变系统
- 更淡的阴影和高光
- 整体色调更轻盈、更梦幻

#### 🎯 背景展示优化
- 大幅降低卡片和分类背景的遮挡
- 背景人物清晰可见
- 保持文字可读性的同时最大化背景展示

## 📝 配置变更

### 新增配置项
```yaml
dream_style: true  # 是否启用梦幻风格增强（默认开启）
```

### 默认值变更
```yaml
# v2.x 默认值
category_opacity: 8
card_opacity: 90

# v3.0 默认值（推荐）
category_opacity: 3   # 降低以展示背景
card_opacity: 70      # 降低以展示背景人物
```

---

## 🎯 使用建议

### 推荐配置（二次元梦幻风格）

#### 有背景图（推荐）
```yaml
theme_color: "#FF88AA"          # 粉色梦幻
background_image: "./images"     # 随机背景图
blur_radius: 15                  # 轻微虚化
category_opacity: 3              # 极淡分类背景
card_opacity: 70                 # 半透明卡片
card_shadow: true                # 启用阴影
```

#### 纯色背景
```yaml
theme_color: "#FF88AA"
background_color: "#FFF0F5"
category_opacity: 10
card_opacity: 90
card_shadow: false
```

### 透明度调节指南

| 场景 | category_opacity | card_opacity | 效果 |
|------|------------------|--------------|------|
| **背景人物清晰可见** | 2-5 | 60-75 | 背景很清晰 ⭐ |
| **平衡效果** | 5-10 | 75-85 | 适度展示 |
| **纯色背景** | 10-15 | 85-95 | 简洁风格 |

## ⚠️ 向后兼容性

### 完全兼容
- ✅ 所有现有配置项保持兼容
- ✅ 旧的透明度配置仍然有效
- ✅ 不影响现有用户的配置
- ✅ 新用户自动使用优化后的默认值

### 升级建议
如果您使用默认配置，升级后会自动应用新的梦幻风格。如果您自定义了透明度，原配置将保持不变。

建议尝试新的默认值以获得最佳效果：
```yaml
category_opacity: 3
card_opacity: 70
```
